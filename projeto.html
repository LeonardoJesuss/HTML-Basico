IDE Recomendada

Inicio

Visual Studio Code
O Visual Studio Code, também conhecido como VS Code, é um editor de código-fonte desenvolvido pela Microsoft. Ele foi lançado em 2015 e rapidamente se tornou um dos editores de código mais populares entre os desenvolvedores. O VS Code é um editor de código-fonte gratuito e de código aberto que suporta várias linguagens de programação, incluindo C++, C#, Java, JavaScript, Python, Ruby e muitas outras. Ele é leve e rápido, e tem recursos avançados que ajudam os desenvolvedores a escrever, depurar e testar seu código de forma mais eficiente. O VS Code tem várias características úteis, como IntelliSense, que fornece sugestões de código enquanto você digita, e a capacidade de depurar o código diretamente no editor. Ele também tem uma grande variedade de extensões disponíveis que adicionam funcionalidades adicionais, como suporte para diferentes linguagens de programação, integração com ferramentas de controle de versão e muito mais. Além disso, o VS Code é altamente personalizável, permitindo que os usuários modifiquem a aparência do editor, adicionem atalhos de teclado personalizados e instalem extensões para personalizar sua experiência de programação. Em resumo, o VS Code é um editor de código-fonte altamente flexível e poderoso que se tornou um favorito entre os desenvolvedores de software em todo o mundo. Com sua ampla gama de recursos avançados e extensões personalizáveis, ele pode ser adaptado para atender às necessidades de qualquer projeto de programação

Inspetor de Elementos

Inicio

inspetor de elementos é uma ferramenta de desenvolvimento de software que está disponível em praticamente todos os navegadores modernos. Ele permite que os desenvolvedores visualizem e manipulem o código HTML, CSS e JavaScript de uma página da web em tempo real. Ao abrir o inspetor de elementos no navegador, os desenvolvedores podem ver a estrutura da página da web e todos os elementos que a compõem. Eles podem editar o código diretamente no inspetor de elementos e ver as alterações sendo aplicadas em tempo real na página da web. O inspetor de elementos também fornece informações úteis sobre os estilos aplicados a cada elemento, permitindo que os desenvolvedores verifiquem se as folhas de estilo estão sendo aplicadas corretamente e façam ajustes necessários.

Estrutura básica do HTML

Inicio

<html>: a tag raiz que envolve todo o documento HTML.
<head>: a seção do documento HTML que contém informações importantes sobre a página, como o título da página, links para folhas de estilo, scripts, entre outros elementos.
<body>: a seção do documento HTML que contém todo o conteúdo visível na página da web, como texto, imagens, vídeos, formulários e outros elementos.
Resumidamente, a estrutura é:
<html>

<head>

<head/>

<body>

<body/>

<html/>

Criando Textos

Inicio

<h1>, <h2>, <h3>, <h4>, <h5>, <h6>: Essas tags são usadas para criar títulos de diferentes níveis de importância em uma página. O <h1> é o mais importante e o <h6> é o menos importante.
<p>: Esta tag é usada para criar parágrafos de texto.
<strong> e <em>: Essas tags são usadas para enfatizar o texto. O <strong> é usado para enfatizar um texto importante, enquanto o <em> é usado para enfatizar uma palavra ou frase em um contexto.
<u>: Essa tag é usada para sublinhar o texto.
<br>: Essa tag é usada para criar uma quebra de linha, sem criar um novo parágrafo.
<blockquote>: Essa tag é usada para citar um trecho de texto longo.
Essas são algumas das tags de texto mais comuns em HTML, mas existem muitas outras! É importante lembrar que as tags de texto devem ser usadas de forma adequada e correta para garantir que o conteúdo da página seja legível e acessível para todos.

Listas ordenadas e não ordenadas

Inicio

As listas são muito usadas em HTML para apresentar informações de forma organizada e fácil de entender. Existem dois tipos de listas em HTML: as listas ordenadas e as listas não ordenadas.
As listas não ordenadas são criadas usando a tag <ul> e seus itens são marcados com a tag <li>. A tag <ul> define o início da lista e a tag <li> define cada item da lista. Por exemplo:
<ul>
<li>Item 1</li>
<li>Item 2</li>
<li>Item 3</li>
</ul>
Isso criará uma lista não ordenada com três itens:
• Item 1
• Item 2
• Item 3

Já as listas ordenadas são criadas usando a tag <ol>, que também define o início da lista, e seus itens são marcados com a tag <li>. A diferença é que os itens da lista ordenada são numerados ou alfabéticos. Por exemplo:
<ol>
<li>Primeiro item</li>
<li>Segundo item</li>
<li>Terceiro item</li>
</ol>

Isso criará uma lista ordenada com três itens numerados:
1. Primeiro item
2. Segundo item
3. Terceiro item

Links

Inicio

Os links são uma parte importante do HTML e são usados para conectar páginas da web, permitindo que os usuários naveguem entre elas. Em HTML, os links são criados usando a tag <a>, que significa "âncora".

A sintaxe básica de um link é a seguinte:

<a href="url-do-destino">texto-do-link</a>
Onde "url-do-destino" é a URL da página para a qual o link aponta, e "texto-do-link" é o texto que aparece no link. Por exemplo, para criar um link para a página inicial do Google, usamos a seguinte sintaxe:

<a href="https://www.google.com/">Visite o Google</a>
Ao clicar nesse link, o usuário será levado para a página inicial do Google.

Além disso, os links também podem ser usados para apontar para locais em uma mesma página. Para isso, utilizamos um identificador (ou "id") na página, e usamos esse id como destino do link. Por exemplo:
<a href="#secao-2">Ir para a seção 2</a>
...
<h2 id="secao-2">Seção 2</h2>
Ao clicar no link, o usuário será levado diretamente para a seção 2 da página.
Os links também podem ter várias propriedades adicionais, como a propriedade "target", que define onde a página de destino será aberta (em uma nova aba, na mesma janela, etc.), ou a propriedade "title", que adiciona um texto descritivo ao link quando o usuário passa o mouse sobre ele.
